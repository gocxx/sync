Detailed Description

template<typename T>
class gocxx::sync::Pool< T >

A thread-safe object pool, similar to Go's sync.Pool.

Designed for efficient reuse of expensive-to-create objects. It stores objects in a stack and reuses them across multiple threads.

Template Parameters
TThe type of objects to store.
